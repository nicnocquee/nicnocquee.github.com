Customize Terminal Prompt with Emoji

Add this line to ~/.bash_profile (Note: You should see the yellow emoji with shade below)

export PS1='\[\e[0;37m\]‚åò \[\e[1;36m\]\W\[\e[1;33m\] <put_emoji_here>  \[\e[0m\]'
